## CATEGORIES

Extract terms into these categories:

### 1. CHARACTER NAMES (人物名称)
- Character names or ways others refer to them
- Proper names of people, immortals, demons, beasts with names
- Output for bare names: Chinese → Pinyin (e.g., "萧炎" → "Xiao Yan", "燕清羽" → "Yan Qingyu")
- IMPORTANT: Extract ALL character names mentioned in dialogue
- **For name+honorific combinations**: Translate the honorific to English, keep name in Pinyin
  - "秦大哥" → "Brother Qin" (NOT "Qin Dage")
  - "李师兄" → "Senior Brother Li" (NOT "Li Shixiong")
  - "王师姐" → "Senior Sister Wang"
  - "张前辈" → "Senior Zhang" or "Elder Zhang"
  - "陈师父" → "Master Chen"
- Common honorifics to translate:
  - 大哥/哥 → "Brother"
  - 大姐/姐 → "Sister"
  - 师兄 → "Senior Brother"
  - 师弟 → "Junior Brother"
  - 师姐 → "Senior Sister"
  - 师妹 → "Junior Sister"
  - 前辈 → Prefer "Senior"; use "Elder" only if the character is formally titled
  - 师父/师傅 → "Master"
  - 真人 → "Venerable"
  - 道友 → "Fellow Daoist"
  - 仙子 → "Fairy" or "Immortal Maiden"
- DO NOT include titles or forms of address alone (师兄, 前辈, etc.)
- DO NOT include generic descriptors (那个人, 老者, etc.)

### 2. CULTIVATION LEVELS (修炼等级)
- ONLY cultivation realms/stages from Dao or Buddhist cultivation systems
- These are formal rank names in the story's power progression
- Output: Chinese → English (e.g., "筑基期" → "Foundation Establishment Stage")
- Examples: 筑基期 (Foundation Establishment), 金丹期 (Golden Core Stage), 斗者 (Dou Zhe)
- NOT generic descriptors: 强大 (powerful), 深厚 (profound), 入微 (subtle)
- NEVER use Pinyin for cultivation levels - always translate to English
- Include stage modifiers (初期, 中期, 后期, 巅峰)

Standard translations you MUST use:
- 后天 → "Postnatal" (NOT "Houtian")
- 先天 → "Prenatal" (NOT "Xiantian")
- 重 → "Layer" (e.g., 一重 → "First Layer", 二重 → "Second Layer")
- 境 → "Realm" (e.g., 神门境 → "Divine Gate Realm")
- 期 → "Stage" (e.g., 金丹期 → "Golden Core Stage")
- 初期/初入 → "Early"
- 中期 → "Middle"
- 后期 → "Late"
- 巅峰 → "Peak"

### 3. SKILLS & ABILITIES (技能招式)
- Named techniques, skills, or formations that characters announce/call out
- These have proper names, not descriptions
- Output: Chinese → English (e.g., "焰分噬天决" → "Flame Devouring Heaven Technique")
- Examples: 焰分噬天决 (Flame Devouring Technique), 七星阵 (Seven Star Formation)
- NOT generic actions: 修炼 (cultivate), 突破 (break through), 凝聚 (condense)
- Translate to evocative English, not Pinyin
- Common suffixes: 决/诀 → "Technique", 功 → "Art", 剑法 → "Sword Art"

### 4. LOCATIONS (地点位置)
- Named places: specific mountains, markets, palaces, cities, realms
- Must be a proper noun/name, not a generic geography word
- Output: Chinese → English (e.g., "迦南学院" → "Canaan Academy")
- Examples: 迦南学院 (Canaan Academy), 青天市场 (Blue Sky Market), 雷霆殿 (Thunder Palace)
- NOT generic: 山 (mountain), 市场 (market), 大陆 (continent)
- Common terms: 宗 → "Sect", 殿 → "Hall/Pavilion", 阁 → "Pavilion"
- 内宗 → "Inner Sect", 外宗 → "Outer Sect"

### 5. ITEMS & ARTIFACTS (物品法器)
- Named items specific to this world that characters use, buy, sell, or discuss
- Must have a proper name or be world-specific
- Output: Chinese → English (e.g., "玄阶丹药" → "Xuan-grade Pill")
- Examples: 灵石 (Spirit Stone), 玄阶丹药 (Xuan-grade Pill), 锁魂链 (Soul-Binding Chain)
- NOT generic objects: 剑 (sword), 刀 (knife), 钱 (money)
- Common terms: 灵石 → "Spirit Stone", 丹药 → "Pill", 法器 → "Artifact"
- Include grade/rank prefixes when present

### 6. ORGANIZATIONS (组织门派)
- Named groups: families, sects, clans, associations, guard units
- Output: Chinese → English (e.g., "云岚宗" → "Misty Cloud Sect")
- Examples: 徐家 (Xu Family), 王族 (Wang Clan), 剑宗 (Sword Sect), 丹药协会 (Pill Association)
- NOT generic terms: 家族 (family), 门派 (sect) alone
- Common suffixes: 宗 → "Sect", 家/族 → "Clan/Family", 门 → "School"
- Titles: 掌门 → "Sect Master", 长老 → "Elder", 太上长老 → "Supreme Elder"

## WHAT TO EXTRACT

ONLY extract terms that are:
- Proper nouns: Character names, place names, organization names
- Specialized vocabulary: Terms unique to this story/genre that need consistent translation
- Recurring terminology: Terms that appear multiple times and require consistency
- In-world concepts: Cultivation levels, techniques, artifacts with specific meanings

World-type guidance ({world_type}):
- xianxia: Cultivation realms, immortal techniques, celestial treasures, sects, dao concepts
- wuxia: Martial arts styles, weapons, jianghu organizations, internal energy concepts
- historical: Court ranks, dynasty terms, official titles, period-specific vocabulary
- modern: Organizations, modern titles, technology terms specific to the story

## EXTRACTION PHILOSOPHY

ONLY extract terms that clearly fit one of the 6 categories above:
- characters: Character names or ways others refer to them
- cultivation: Cultivation realms/stages (Dao/Buddhist cultivation systems)
- skills: Named techniques, skills, or formations that characters call out
- items: Named items specific to the world (pills, artifacts, currencies)
- locations: Named places (mountains, markets, palaces, cities)
- organizations: Named groups (families, sects, clans, associations)

**THE CATEGORY TEST:** Before extracting, ask: "Which category does this fit?"
- If it fits one of the 6 → EXTRACT
- If it doesn't fit any → DO NOT EXTRACT

When in doubt, OMIT. A lean glossary is better than a bloated one.

## COMPOUND TERM HANDLING

When a compound term contains a named entity + generic descriptor, extract only the named entity:

- 云岚宗护法 → Extract: 云岚宗 (the organization), Skip: 护法 (generic role)
- 镇宗祖器 → Extract: 祖器 (named item type), Skip: 镇宗 (generic descriptor)

The named entity is what fits a category; the generic part doesn't.

## WHAT TO AVOID

Do NOT extract:
- Complete phrases or dialogue (e.g., "定其形 困其神 终其命" is a chant, not a term)
- Generic vocabulary that doesn't need special translation
- One-off expressions unlikely to recur
- Self-explanatory compound words
- Mantras, chants, or poetry lines (unless they're named techniques)
- Common martial arts actions (出手, 接招, 过招)
- Generic pronouns: 他, 她, 你, 我, 我们
- Common address forms used generically: 师兄, 师姐, 前辈, 师父 (unless part of a specific name)
- Generic insults: 废物, 蠢货, 混蛋
- Common actions: 走, 来, 去, 说, 看
- Numbers and quantities
